教学资源与作业管理平台 · 使用说明

系统简介

教学资源与作业管理平台是一款面向学校的在线教学管理平台，支持多教师、多班级协同工作，提供作业发布、学生提交、文件管理和教学资源共享等完整功能。

角色说明
教师管理班级、创建作业、分发教学资源
学生查看作业任务、上传作业文件、下载共享资源

教师使用指南 · 账号与登录

注册新账号

  1. 访问教师端页面，点击"注册"
  2. 填写用户名、真实姓名、密码
  3. 真实姓名必须在教师白名单中，否则注册失败，请联系管理员确认

修改个人密码

  1. 点击页面右上角的"修改密码"按钮
  2. 输入原密码
  3. 输入新密码（至少 6 位）并确认
  4. 点击确认完成修改

班级管理

创建班级

  1. 点击"创建班级"按钮
  2. 输入班级名称
  3. 确认创建

删除班级

  1. 在班级列表中点击"删除班级"
  2. 确认操作

删除后班级进入历史记录，教师端不再显示。

学生管理

批量添加学生

  1. 在班级列表中点击"学生管理"
  2. 点击"批量添加"
  3. 按以下格式输入学生信息（每行一个）：
学号 姓名
20240101 张三
20240102 李四
20240103 王五

可选填写初始密码：20240101 张三 mypassword

  1. 点击提交

删除学生

  • 单个删除：点击学生信息右侧的"删除"按钮
  • 批量删除：勾选多个学生后，点击"批量删除选中学生"

重置学生密码

点击对应学生的"重置密码"按钮，输入新密码即可。

作业管理

创建班级作业

  1. 在班级列表中点击"作业管理"
  2. 点击"创建新作业"
  3. 填写以下信息：
    • 作业名称（必填）
    • 上传密码（系统会自动生成，也可自定义）
    • 截止时间（必填）
    • 作业描述（可选，填写作业要求、注意事项等）
  4. 点击"创建"

系统会自动检测密码是否与其他有效作业重复，避免学生上传到错误的作业。

创建临时作业

适用于没有预先建立班级名单的临时情况，学生使用”快速上传“功能上传作业，不需要登录，上传时输入学号。

  1. 点击"创建临时作业"
  2. 填写作业名称、密码、截止时间（作业描述可选）
  3. 点击创建

临时作业文件命名格式为 学号.扩展名，后续可在"关联班级"操作后进行学号匹配。

关联临时作业到班级

  1. 在临时作业列表中，点击对应作业的"关联班级"按钮
  2. 选择目标班级
  3. 确认关联

编辑作业

  1. 在作业列表中点击"编辑"按钮
  2. 可修改：作业名称、上传密码、截止时间、作业描述
  3. 点击保存

修改后立即生效，学生使用旧密码将无法继续上传。

查看提交情况

点击作业列表中的"查看提交"：

  • 班级作业：显示班级所有学生的提交状态（已提交/未提交）、提交时间和文件名
  • 临时作业：显示所有已提交的学号和文件信息

批量下载作业

  • 点击作业列表中的"批量下载"按钮
  • 或在"查看提交情况"页面点击"批量下载所有作业"
  • 系统自动将所有已提交文件打包为 ZIP 下载

删除作业

  1. 点击"删除"按钮
  2. 在确认对话框中确认

⚠️ 警告：删除作业会永久删除所有相关记录和已提交的文件，不可恢复。

教学资源管理

教学资源模块提供类似文件浏览器的界面，用于上传和管理教学资料。

上传文件

  1. 进入"教学资源"模块
  2. 点击"上传文件"
  3. 选择本地文件并确认

创建文件夹

  1. 点击"新建文件夹"
  2. 输入文件夹名称
  3. 确认创建

重命名文件/文件夹

  1. 找到目标文件或文件夹
  2. 点击"重命名"按钮
  3. 输入新名称并确认

删除资源

  1. 找到目标文件或文件夹
  2. 点击"删除"按钮并确认

删除前系统会检查是否有关联的共享，如有关联请先删除共享再操作。

资源共享

口令共享（任何人可访问）

  1. 在"资源共享"模块中点击"创建共享"
  2. 选择要共享的文件或目录
  3. 设置共享信息：
    • 共享名称
    • 共享码（系统自动生成，可自定义，支持字母和数字）
    • 描述（可选）
    • 过期时间（可选）
  4. 点击"创建"

生成共享码后，将共享码发给任何需要访问的人，对方访问 /share.html 页面输入共享码即可下载。

班级共享（仅指定班级学生可访问）

  1. 点击"创建班级共享"
  2. 选择目标班级
  3. 选择要共享的资源和填写描述
  4. 点击创建

班级学生登录学生门户后，在"共享资源"页面可以看到并下载。

删除共享

在共享列表中点击"删除"即可，不影响原始文件。

学生使用指南 · 快速上传（无需登录）

适用于临时作业，或不需要学生账号的场景。

  1. 访问快速上传页面（/student.html）
  2. 输入教师提供的上传密码，点击"验证"
  3. 验证成功后：
    • 班级作业：输入学号，系统验证学号是否在班级名单中，通过后显示姓名
    • 临时作业：无需验证学号，直接跳到上传步骤
  4. 可查看作业描述（如有）
  5. 选择文件（支持拖拽上传）
  6. 点击"上传"

注意事项

  • 超过截止时间后，上传密码将失效，无法继续提交
  • 文件自动重命名：
    • 班级作业：学号_姓名.扩展名（例：20240101_张三.pdf）
    • 临时作业：学号.扩展名（例：20240101.pdf）
  • 如果已上传过文件，系统会提示是否覆盖，确认后新文件将替换旧文件

学生门户（需要登录）

提供更完整的功能，包括查看所有作业状态和历史上传记录。

默认密码

  • 学号：由教师导入
  • 初始密码：123456（或教师设置的密码）

登录

  1. 访问学生门户页面（/student-portal.html）
  2. 输入学号和密码
  3. 点击"登录"（或按回车）

我的作业

登录后，"我的作业"页面显示所有班级作业：

字段说明
作业名称点击可查看作业描述
截止时间超过截止时间显示"已截止"
状态进行中 / 已截止
提交状态已提交 / 未提交
操作未截止时显示"上传作业"按钮

上传作业

  1. 点击"上传作业"按钮
  2. 在弹出框中选择文件（支持拖拽）
  3. 点击"上传"

共享资源

"共享资源"页面显示教师发布的班级共享资料，点击"下载"即可获取。

我的上传

"我的上传"页面显示历史上传记录：

  • 文件名、所属作业、上传时间
  • 点击"下载"可重新下载已提交的文件

修改密码

  1. 点击左下角的"修改密码"按钮
  2. 输入原密码
  3. 输入新密码（至少 6 位）并确认
  4. 点击"确认修改"

退出登录

点击左下角"退出登录"按钮。

共享访问

任何人（无需账号）均可通过共享码下载教师分享的资源。

访问步骤

  1. 访问共享访问页面（/share.html）
  2. 输入教师提供的共享码（不区分大小写）
  3. 点击"访问共享内容"（或按回车）

下载文件

根据共享类型，下载方式不同：

共享类型下载方式
单个文件点击"下载文件"按钮
目录浏览目录，可单独下载文件，也可勾选多个后"下载选中"，或"下载全部"打包下载
多个文件同目录方式
作业包点击"下载文件"按钮

共享信息说明

  • 访问次数：当前访问次数（若有上限则显示 已访问/上限）
  • 过期时间：共享的有效期（若已过期则无法访问）
  • 描述：教师添加的说明信息

目录浏览

  • 点击文件夹名称可进入子目录
  • 顶部面包屑导航可快速返回上级目录
  • 支持全选后批量下载

常见问题

Q：忘记上传密码怎么办？
联系教师获取当前有效的上传密码。
Q：上传密码提示已过期？
作业已截止，无法继续提交。如需延期，请联系教师修改作业截止时间。
Q：学号验证失败？
确认输入的学号是否正确，或联系教师确认学号是否已录入班级名单。
Q：学生门户初始密码是多少？
默认初始密码为 123456，或教师导入时设置的密码。如忘记密码，请联系教师重置。
Q：上传后文件名变了是正常的吗？
是的。系统会自动将文件重命名为标准格式（学号_姓名.扩展名），方便教师管理和识别。
Q：共享码不区分大小写吗？
是的，共享码访问时不区分大小写，可放心输入。
Q：教师注册失败，提示"姓名不在白名单"？
请联系管理员将您的真实姓名加入白名单后，再重新注册。
